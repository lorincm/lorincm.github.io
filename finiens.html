finiens

What's finiens?

Finiens is a minimalist geolocator game where you must point your phone (literally) where the given settlement is compared to you

play it alone, or with friends, it's a good time-killer and party game

how does it work?

the game needs acces to your phones sensors. it uses your GPS and Accelerometer and calculate your device's heading

the following code snippet defines the function that calculates the bearing angle between your position and the game-given settlement

		double bearing(double lat1, double long1, double lat2, double long2) //1: device location 2: town
		{ 
			double dLon = (long2 - long1);
			double y = sin(dLon) * cos(lat2);
			double x = cos(lat1) * sin(lat2) - sin(lat1)*cos(lat2) * cos(dLon);

			double heading = atan2(y, x);
		  
			heading = heading * 180 / d_pi; 
			heading = (heading + 360) % 360;
			return heading;
		}
